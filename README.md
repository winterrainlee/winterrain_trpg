# Overview
> 당신이 만드는 세계 속에서, 당신의 목표를 향해 달려가세요
ChatGPT 5.1 thinking 환경에서 작동하는 턴제 텍스트 롤플레잉 게임 프롬프트입니다.
커스텀GPT 혹은 프로젝트에서 사용할 수 있습니다.

last version: **1.4.3 (251120)**

## 아주 빠른 사용법
`시작`을 입력하여 새로운 세션을 시작합니다.
세계와 플레이어 캐릭터(PC) 설정 후 `프롤로그 시작`을 통해 이야기가 시작됩니다.
선택지 번호를 입력하거나, 당신의 문장을 입력하여 PC의 행동을 지시하면 다음 턴으로 넘어갑니다.
20면 주사위+보정치가 행동 난이도(DC, 10~22) 이상일 경우 성공으로 판정되며, 판정 결과에 따라 세계의 반응과 변화가 달라집니다.

`상태 보기`를 통해 PC와 세계를 확인할 수 있습니다.

PC의 장기 목표가 달성되거나, 사망할 경우(HP=0) 세션이 종료됩니다.
도중에 세션을 마치고 싶다면 `END`를 입력해 주세요.

자세한 내용은 `도움말`로 물어보세요!

## 파일 구성
* Core.md: 커스텀GPT 혹은 프로젝트의 메인 지침으로 입력
* Setup.md: 첨부 파일, 설정 모듈 (세계관 및 플레이어 캐릭터**PC** 생성)
* StatusView.md: 첨부 파일, 상태창 모듈
* Ending.md: 첨부 파일, 엔딩 모듈
